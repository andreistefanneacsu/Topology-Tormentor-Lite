# Topology Tormentor Lite

## Descriere
**Topology Tormentor Lite** este o aplicație de simulare a rețelelor de calculatoare care permite utilizatorilor să creeze, să vizualizeze și să testeze diferite topologii de rețea într-un mediu virtual. Aplicația este concepută în special pentru studenți și începători care doresc să învețe concepte de bază din domeniul rețelisticii.

Programul oferă o interfață grafică intuitivă prin care utilizatorii pot adăuga dispozitive de rețea, le pot conecta între ele și pot observa modul în care datele circulă în rețea.

## Funcționalități
- Crearea și editarea topologiilor de rețea
- Adăugarea dispozitivelor precum PC-uri, routere și switch-uri
- Conectarea dispozitivelor prin cabluri virtuale
- Simularea comunicării între dispozitive
- Interfață grafică simplă și ușor de utilizat

## Scop
Scopul aplicației este de a oferi un mediu de învățare pentru concepte precum:
- adresarea IP
- rutarea pachetelor
- conectivitatea între dispozitive
- structura topologiilor de rețea
